# Project Template

A small repository that will show how to call stata from R.

# How to use

1.  Clone
2.  Adjust the my_projdir paths in:
    1. /R_code/project_logistics/R_paths_libraries.R
    2. /R_code/project_logistics/R_paths_libraries_setup.R
    3. /R_code/stataR_wrapper.R
    4. /stata_code/test1.do
3. Run   "/R_code/stataR_wrapper.R"
